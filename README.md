# Sabrina-Bot
A simple discord bot built in Python 3.13 w/ Pycord to generate Sabrina Carpenter memes via command.

## Running the bot
1. Clone the repository and initialize a virtual environment
2. Run `pip install -r requirements.txt`
3. Rename the `.env-example` file to `.env` and paste your discord bot token / optional Guild ID
4. Run `python bot.py`
